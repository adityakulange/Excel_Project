# Employee-Target-Data-Analysis (Interactive Dashboard creation using MS Excel)

## Project Overview

This project focuses on analyzing sales performance data of sales executives across multiple regions to evaluate target achievement, identify high-performing regions and employees, and uncover sales trends over multiple days.
The analysis helps management track productivity, measure efficiency, and make data-driven business decisions.

## Dataset used
- <a href="https://github.com/adityakulange/Excel_Project/blob/main/Employee%20Target/Employee%20Target%20Data.xlsx">Employee-Target-Dataset</a>

## Questions (KPIs)

1. Calculate Total Transaction by each State
2. Calculate Total Transaction by each State as well as Product Category
3. Calculate total quantity Sold by Region
4. Calculate total quantity Sold by Region as well as Product name
5. Calculate the average sales amount of each state
6. Calculate the average sales amount of each state as well as Product name
7. Calculate Each city max sales.

## Process

1.	Data Collection
•	Imported Excel dataset into Python using Pandas
2.	Data Understanding
•	Checked data types, columns, and structure
•	Verified total sales calculations
3.	Data Cleaning
•	Removed unnecessary columns (if any)
•	Ensured numeric consistency
•	Handled missing or inconsistent values
4.	Exploratory Data Analysis (EDA)
•	Region-wise total and average sales
•	Employee-wise performance analysis
•	Day-wise sales trend analysis
5.	Performance Metrics Calculation
•	Total sales by region
•	Average sales by region
•	Target hit and gap analysis
•	Identification of top and bottom performers
6.	Insights & Interpretation
•	Discovered regions with highest sales contribution
•	Identified sales executives exceeding targets
•	Analyzed daily sales behavior
7.	Visualization (Optional)
•	Bar charts for regional sales
•	Line charts for daily trends
•	KPI indicators for target achievement
